# BAJA-SAE-Chassis-Analysis
Conducted FEA of a BAJA SAE chassis to evaluate strength, stiffness, and vibration safety. Simulated frontal, side, rear, rollover, and torsional loads using ANSYS. Achieved FOS ≈ 2 with stress 238–464 MPa. Optimized beam profiles and materials for improved durability and SAE compliance.
